requirements :
python version 3 or upper (better for your performance)

HOW TO USE THE CODE AS A MODULE?
you can use the pysize.py as a module like you install, but it is a bit different.
first, install the pysize.py.
second, open your python directory, it must have the version at the end of the folder.
finally, you can open "lib" folder and paste the pysize.py, use the module!

main parameter "linux" :
for windows/mac to get the size of a file, is different for linux. if you are using windows or mac, you can
write main parameters but if you are using linux, do this :
```
function_you_are_using_is_here(file_or_path, linux=True)
```

you must set 'linux' on 'True'.
